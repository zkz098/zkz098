# 开始前的准备

# 游戏本体

首先，双方需要使用一样的游戏版本和 ModLoader
推荐直接将游戏打包后发给对方

玩原版也需要安装 Forge/Fabric (后续步骤需要使用)

如果需要安装模组，需要查看模组安装需求 (建议在 MCMOD 百科查看)
分为以下几种情况：

  1. 客户端需装，服务端需装
  2. 客户端选装，服务端需装
  3. 客户端需装，服务端选装
  4. 客户端需装，服务端无效
  5. 客户端无效，服务端需装

1.2.3 情况建议双方都安装此模组

2. 情况主机端必装，其他人不装可能有影响
3. 情况所有人最好都装

4.5 情况无效端不需要装

那么，什么是服务端和客户端？
假设 S 为存档拥有者，C 为加入的玩家
即 S 为服务端 + 客户端，C 为客户端

# 游戏模组

版本截至 2022-1-13 时
安装对应版本的模组 (服务端，客户端均需)：

  • 1.15.2-1.18.1(forge/fabric)
  • MCMOD 百科
  • modrinth
  • 1.12.2-1.17.1(forge)/1.14.4-1.17.1(fabric)
  • MCMOD 百科
  • CurseForge
  • 1.7.2-1.12.2(forge)
  • MCMOD 百科
  • CurseForge,

如果你是离线登录，需要将 OnlineMode 更改为 false , 具体配置文件更改请参阅 MCMOD 百科
建议安装第一个模组，有 GUI 页面

# IPV6

访问 ipv6test 来检测双方的 IPV6
如果一切正常，你看到的页面应该类似于这样：

底部应该是 10/10, 作者的 DNSv6 不太正常
然后检查 IPV6 的 IP, 以下有几个 IP 开头的意义：

  • fe80 开头的无法用于上网，作用就是有个 IP
  • 2001 开头为 teredo, 不建议用于联机
  • 240e、2409、2408 为国内三大运营商的 IPV6, 非常正常
  • 其他开头请见下一步

如果属于 fe80 或 2001 开头建议使用 内网穿透 而非 IPV6
内网穿透短期不考虑出教程
