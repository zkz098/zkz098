# int 型

python 中的 int 不分长短型，不可迭代 (不可被 for 遍历)

从 float 到 int 是向下取整，1.99->1

# float 型

python 中的 float 约等于 C 中的 long double, 无位数限制，不可迭代

float 有不确定小数后缀，最好不要使用 == 比较

# str 型

str 型为字符串，可迭代，长度不限
