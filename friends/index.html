# 基本规则

本站原则上只允许个人博客类网站交换友链，且不得违反下列硬性要求:

  1. 网站不能存在违反中国法律的内容
  2. 网站不能是纯广告性质 (本人主观判断) 的网站
  3. 网站不能存在大量令人不适的内容
  4. 不能是盗版资源 (或有争议的资源) 为主的资源站

一些原则性质的要求原则就是用来打破的，但如果您的网站非常的令人我喜欢，那这些规则不适用于您:

  • 您的网站最后一篇文章 / 最近动态应该在 3 个月内
  • 应该以有实际意义的文章为主
  • 您的页面在天津电信环境下加载不应长于 7s
