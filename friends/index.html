# 友链交换规则

# 基本规则

本站原则上只允许个人博客类网站交换友链，且不得违反下列硬性要求:

  1. 网站不能存在违反中国法律的内容
  2. 网站不能是纯广告性质 (本人主观判断) 的网站
  3. 网站不能存在大量令人不适的内容
  4. 不能是盗版资源 (或有争议的资源) 为主的资源站
  5. 转载文章比例不得高于 30 %

同时，我希望满足如下要求：

  1. 网站在本人所在位置（可查看 github 主页）访问畅通
  2. 网站已稳定运行不少于三个月
  3. 网站内容应以有实际意义的文章为主

对于长时间无回复的请求，您可尝试再次发送请求或更换可信度更高的邮箱。
