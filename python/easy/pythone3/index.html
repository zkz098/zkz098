# 逻辑与比较运算符

# 逻辑运算符

在 python 中，逻辑运算符有以下几种：

符号英文 作用
and 两真为真，一假为假
or 一真为真，两假为假
not 非真即假

python 中的逻辑运算符有优先级，not > and > or, 括号可以改变优先级

# 运算符

基本运算符：

符号 作用 举例 符号 作用 举例
+ 相加 1+1=2 += 自增操作 i += 1(i=i+1)
- 相减 2-1=1 -= 自减操作 i -= 1(i=i-1)
* 相乘 2*3=6 ** 乘方 2**3=8
/ 相除 3/2=1.5 // 整除 3//2=1
% 求余 3%2=1 *= 自乘操作 a *= 2

# 比较运算符

符号 作用 符号 作用
> 比较大于 < 比较小于
>= 大于等于 <= 小于等于
== 相等 != 不等

比较运算符返回值均为 bool 类型，非真即假 部分类无法进行比较
