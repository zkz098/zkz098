# hexo-indexnow 插件

此插件仅提供了 indexnow 链接推送功能，建议搭配其他插件使用

目前只有 1.0.6+ 和 1.0.3 可以正常运行，请勿安装 1.0.5
1.0.3+ 均有 fetch 兼容支持
