资源地址格式如下:

  • https://example.com/xxx.js 使用 http (s) 地址加载 js
  • combine/xxx.js,xxx.js 使用 jsdelivr 的 combine 功能加载 (不推荐)
  • npm/xxx/xxx.js 使用 npm 源加载 js
  • gh/xxx/xxx.js 使用 gh 源加载 js
  • xxx.js 从本地加载 js
    优先级如下:
    http>combine>npm>gh>本地
